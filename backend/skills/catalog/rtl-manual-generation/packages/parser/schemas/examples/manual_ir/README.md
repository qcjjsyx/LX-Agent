# Manual IR Examples

这个目录放的是 `Manual IR` 的示例文件，用来帮助理解每个字段的用途。

说明：

- 这里的示例主要用于 schema 审查和字段语义说明。
- 示例字段名和对象关系应被视为候选设计，而不是当前 parser 已自动生成的最终产物。
- 若某些字段与当前项目中的真实模块命名、实际可推导语义存在偏差，应以“帮助理解 schema”为主要目的来阅读。
